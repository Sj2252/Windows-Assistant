Command Manual

🚀 Activation

  • "Arise": Wakes the assistant from sleep mode.
  • "Iris": Use this prefix before any command (e.g., "Iris, what time is it?").

💻 Media & System

  • Volume: "Set volume to 50 percent"
  • Brightness: "Set brightness to 80 percent"
  • Media: "Play music", "Pause", "Next song", "Previous"

📂 Applications

Supported: Chrome, Spotify, Excel, Word, Notepad, Calculator, VLC, and more.

  • "Open Spotify" / "Close Spotify"
  • "Maximize Notepad" / "Minimize Notepad"

🔍 General

  • "Search for [query]": Opens browser search
  • "What time is it?": Speaks current time
  • "Go to sleep": Returns to dormant mode
  • "Stop Assistant": Closes the assistant entirely

📂 Compatible Apps

You can control these apps by saying "Iris [open/close] [name]"
